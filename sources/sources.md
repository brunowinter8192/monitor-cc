## Notes

- Domain shows the URL host (or `github.com/<org>` for repo references).
- Type values: `Repo` | `Web` | `Paper` | `Thesis` | `Forum`.
- Status values: `Referenced` (mentioned/consulted, not RAG-indexed) | `Verified` (URL + content manually confirmed) | `Indexed (RAG: <collection>)` (live in the named RAG collection).
- API Mirror rows aggregate multiple local mirror files of the same Anthropic docs topic. Files live under `Meta/ClaudeCode/MCP/RAG/data/documents/Monitor_reference/` and are indexed in the `Monitor_reference` collection. Per-file decision mapping is not maintained — these are generic API documentation excerpts, hence `Decision Steps: misc`.
- Forum sources from plugin searches stay `Referenced` permanently (no RAG index).
